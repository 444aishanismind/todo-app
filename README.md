
# Todo App

A simple Todo application built with Node.js, Express, EJS, and MongoDB.

## Features
- Create todos
- Update todos
- Delete todos
- Server-rendered UI with EJS

## Tech Stack
- Node.js
- Express.js
- EJS
- MongoDB Atlas
- Render

## Live Demo
https://todoit-ayfp.onrender.com

## Run Locally
1. Clone the repository
2. Install dependencies:
   npm install
3. Create a `.env` file:
   PORT=8000
   MONGODB_URI=your_mongodb_uri
4. Start the app:
   npm start
